# Guess Who — audio clips

Drop the puzzle tracks here, then reference them from `guesswho-clips.js`.

**Day 1 needs:** `love-supreme.mp3` (John Coltrane — *A Love Supreme, Pt. 1: Acknowledgement*).

Use recordings you own. The game plays the file straight through; when the track
ends (or the player uses all 3 lives / names everyone), it reveals the answers.
